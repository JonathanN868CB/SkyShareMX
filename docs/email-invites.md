# Email invitations (deprecated)

SkyShare now relies entirely on Google sign-in for onboarding. Any teammate with an `@skyshare.com` address can visit the portal, click **Continue with Google**, and land in the app as a Viewer. The Netlify invitation function and SMTP configuration have been retired, so no additional setup is required.

Keep this file for historical reference only. If a future project reintroduces outbound invitations, document the new flow and required environment variables here.
